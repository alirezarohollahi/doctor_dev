
# Doctor Dev lab test pack

Overlay this pack on top of the latest Doctor Dev project zip. It adds local lab env files, a colored 4-port TCP listener, a TCP sender, and helper scripts to run:

- Panel on `9000`
- Node A on `9001`
- Node B on `9002`
- Listener ports `9101-9104`

Read `docs/LAB_ROUTING_TEST_GUIDE.md`.
